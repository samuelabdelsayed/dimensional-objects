Dimensional Objects Screenshot Tool

Instructions for capturing dimensional object images

This tool helps you capture screenshots of dimensional objects for the README. Follow these steps to create images for each dimensional representation of a cube:

Steps to capture images:

1. Open the main application

First, ensure the main application is running on http://localhost:3001

Open Main Application
2. Set up each dimension for screenshot

For each dimension:

  • Select "cube" from the dropdown menu
  • Set the color to something visually appealing (e.g., blue or purple)
  • Position your browser to frame just the specific dimension's display
  • Take a screenshot (Command+Shift+4 on Mac, or use your preferred screenshot tool)
  • Save the screenshot as "1d_cube.png", "2d_cube.png", "3d_cube.png", or "4d_cube.png"
3. Save screenshots to the images directory

Save all screenshots to the "images" directory in the project root

4. Verify images in README

After capturing all images, restart the server and check the README to ensure images appear correctly

What each dimension should showcase:

  • 1D Cube: A horizontal line segment
  • 2D Cube: A square
  • 3D Cube: A standard 3D cube with visible faces
  • 4D Cube: A tesseract (cube within cube with connecting lines)
